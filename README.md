# ssha
secure hash utility

Add repository:

```xml
<repositories>
  <repository>
    <id>drwolf maven public</id>
    <url>https://maven.drwolf.it</url>
  </repository>
</repositories>
```

Add dependency:

```xml
<dependency>
  <groupId>com.github.drwolf-oss</groupId>
  <artifactId>ssha</artifactId>
  <version>1.0.0</version> 
</dependency>
```
